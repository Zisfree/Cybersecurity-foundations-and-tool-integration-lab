# How does a thin wire support this many connetions worldwide?

### For a while I wasn't even able to comprehend that millions of data traveling to and fro at the same millisecond without colliding but now its way better. This is how all of it works:
- First of all the data we send travels in an optical fibre cable which has number of strands of glasses.
- How does this glass contain data? Just like how data can be converted into 0's and 1's in form of electricity the same happens here using wavelength-division multiplexing
- This is actually division of data into multiple colors with 100+ different wavelenghts.(Sooo cool right?)
- But how can this data not collide with other's data which is also traveling within same underwater cable? This actually is prevented by converting our data which is in colors into small packets.
- These packets are transfered one at a time-interval that respects other data's too.
- So the prblem solved!
