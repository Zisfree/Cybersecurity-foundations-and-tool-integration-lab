# What is API? Fundamentaly

## Birth
- In 1960's there were rinters and for a printer they had to write code on how much ink or how fast to roll the paper.
- Also now there were different brands and for each brand you had to write a different code which was frustrating.
- So engineers mde a universal code for them that works for all printers.

## API's use in worlds largest WAN
- Now, in 2000's when big companys selling stuff had huge databases other companies wanted to use this database too and to to that they need to connect to them.
- But, if they would connect then it might leak those databases.
- To prevent this they coded API into internet using it like a wall to protect databases and made a small secure hole through which their data could be transfered to the ones in need.
- You can also think API as a plug to connect databases from other sites. `server -> websites`

## WinSock
- Similar to API, WinSock was developed for windows cause it had the same problem of modems instead of printers.
- They had to write codes for diff modems so they made WinSock which was a plug to connect it to Internet all all sorts of devices.
- Atlast google also used that same plug.
