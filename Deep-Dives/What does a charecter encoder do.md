## Charecter encoder:
- Charecter encoders are mediators between us and the system.
- Since the system doesn't understand our alphabets, we need some language to be able to converse.
- To do that we use charecter encoders. Like if I use an ASCII encoder it will tell the system whatever I type in ASCII charecters. After the system understands it, it sends us the output in form of ASCII which is converted back to readable text before being shown.
- These charecters can be different and work differently for different systems. Like if I sned an emojie ASCII wont understand it cause it only reads normal aplhabets.
- So we use different kinds of charecter encoders for different reasons.
