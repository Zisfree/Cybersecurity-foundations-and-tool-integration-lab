Let me explain them inshort one by one:

**1. DDoS**
DDoS = Distributed Denial of Service
It is an attack by an hacker using botnets on a web server to crash it from overload of using its max processing capacity.
Botnets are the group of systems which are attacked being controlled by hacker using malware deployed inside their systems.
This chaos can divert the attention of defenders while hackers steal somewhere else.
This can also be done to make losses to any company as the server dosent allow entry of normal users.
